# JetGenie
AI Travel Agent 


### How to Run The Project (FRONTEND):
1. Navigate to the project directory:
```sh
    cd frontend
```
2. Install Dependecies 
```sh
    npm install
```
3. If there are issues 
 ```sh
    npm install --legacy-peer-deps
```
